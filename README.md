# laoluo

> 罗永浩播客主持人 Skill for WorkBuddy

用老罗的风格做播客——开场、访谈、评论、单口，全套。

## 安装

将 `laoluo` 文件夹放入 WorkBuddy 用户技能目录：

```bash
~/.workbuddy/skills/laoluo/
```

或克隆到本地后：

```bash
git clone https://github.com/YOUR_USERNAME/laoluo.git ~/.workbuddy/skills/laoluo
```

## 使用方法

在 WorkBuddy 中触发：

```
扮演罗永浩做播客主持人
```

或指定场景：

```
帮我写一期播客开场，关于[主题]
用老罗风格主持一期[嘉宾]访谈
生成一期关于[话题]的单口评论
```

## Skill 结构

```
laoluo/
├── SKILL.md              # 核心人格定义 + 播客框架
├── references/
│   └── laoluo-wisdom.md  # 经典语录、修辞框架、话术模板
└── README.md
```

## 主要功能

- 播客开场白生成（三种风格：哲学歪楼 / 自嘲 / 争议预判）
- 嘉宾访谈主持（破冰 → 张力 → 深挖 → 释放）
- 话题评论单口（六段式结构）
- 老罗风格的金句生成
- 过渡语与收尾模板
